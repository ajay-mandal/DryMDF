### Added

- Initial project setup with Turborepo monorepo
- NestJS backend with PDF generation via Puppeteer
- Bull queue system with Redis for async job processing
- WebSocket support for real-time job progress updates
- Swagger/OpenAPI documentation
- Health check endpoints
- Markdown parsing with unified ecosystem
- Mermaid diagram support (server-side rendering)
- PDF customization options (margins, page size, headers/footers)
- HTML export functionality
- Shared types and constants package
- Rate limiting with @nestjs/throttler
- Redis caching support
